# Алгоритмический мини-пакет

Проект, реализующий коллекцию классических алгоритмов сортировки, генераторов тестовых данных и инструментов для бенчмаркинга.

## Возможности

### Алгоритмы сортировки

- **Bubble Sort** — пузырьковая сортировка
- **Quick Sort** — быстрая сортировка с поддержкой отрицательных чисел
- **Counting Sort** — сортировка подсчётом для целых чисел
- **Radix Sort** — поразрядная сортировка с поддержкой отрицательных чисел
- **Bucket Sort** — блочная сортировка для вещественных чисел
- **Heap Sort** — пирамидальная сортировка

### Генераторы тестовых данных

- `rand_int_array()` — случайные целые числа
- `rand_float_array()` — случайные вещественные числа
- `nearly_sorted()` — почти отсортированный массив
- `many_duplicates()` — массив с множеством дубликатов
- `reverse_sorted()` — массив в обратном порядке

### Инструменты

- **Benchmarking** — замер производительности алгоритмов
- **Stack** — реализация структуры данных стек
- **Fibonacci & Factorial** — вычисление чисел Фибоначчи и факториала

## Установка и запуск

## Особенности реализации

### Типизация

Все функции имеют строгую типизацию и проверку типов входных данных:

```python
def bubble_sort(a: list[int]) -> list[int]:
    """Пузырьковая сортировка по возрастанию"""
    if not type_check_int(a):
        raise ValueError("Все элементы списка должны быть типа 'int'")
    # ...
```

### Поддержка отрицательных чисел

`radix_sort` и другие алгоритмы корректно обрабатывают отрицательные числа:

```python
nums = [5, -3, 8, -1, 0, -7]
result = radix_sort(nums)
# [-7, -3, -1, 0, 5, 8]
```

### Воспроизводимость результатов

Генераторы поддерживают параметр `seed` для воспроизводимых результатов:

```python
# Одинаковые результаты при одном seed
arr1 = rand_int_array(10, -100, 100, seed=42)
arr2 = rand_int_array(10, -100, 100, seed=42)
assert arr1 == arr2
```
